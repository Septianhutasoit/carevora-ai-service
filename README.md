# Career AI Service — FastAPI Intelligence Layer

**Career AI Service** adalah komponen inti kecerdasan buatan (AI) dalam ekosistem **CareerMatch**. Layanan ini berfungsi sebagai **jembatan komunikasi** antara Backend (NestJS) dan Model AI (Hugging Face), serta menyediakan kemampuan komputasi vektor dan semantic search untuk mendukung rekomendasi karier yang akurat dan personal.
